# Quick Reference Guide

## File Overview

### Python Scripts

- **generate_mendix_enums.py** - Converts CSV → Mendix SDK TypeScript (for importing)
- **generate_export_script.py** - Generates the TypeScript export module

### TypeScript Files

- **export_mendix_enums.ts** - Export module (generated by generate_export_script.py)
- **example_export.ts** - Complete example of using the export function
- ***_mendix_enums.ts** - Generated import code (created by generate_mendix_enums.py)

### CSV Templates

- **enumerations_simple.csv** - Single language example (English only)
- **enumerations_template.csv** - Multi-language example (EN, FR, ES)

## Common Commands

```bash
# Step 1: Set your Personal Access Token
export MENDIX_TOKEN='your-personal-access-token'
```

### Import (CSV → Mendix)

```bash
# Generate import code
python generate_mendix_enums.py my_enums.csv

# This creates: my_enums_mendix_enums.ts
# Use in your Mendix SDK project
```

### Export (Mendix → CSV)

```bash
# Step 1: Generate the export module (if not already done)
python generate_export_script.py

# Step 2: Use in TypeScript
npx ts-node your-export-script.ts
```

## CSV Format Quick Reference

### Minimal (single language)

```csv
EnumName,ValueName,Caption,LanguageCode
Status,Active,Active,en_US
Status,Inactive,Inactive,en_US
```

### Multi-language

```csv
EnumName,ValueName,Caption,LanguageCode
Status,Active,Active,en_US
Status,Active,Actif,fr_FR
Status,Active,Activo,es_ES
Status,Inactive,Inactive,en_US
Status,Inactive,Inactif,fr_FR
```

## Troubleshooting

### Import Issues

- **CSV format error**: Check that header is exactly: `EnumName,ValueName,Caption,LanguageCode`
- **Python error**: Ensure Python 3.6+ is installed
- **File not found**: Use full path or ensure you're in the correct directory

### Export Issues

- **Connection error**: Verify Mendix API credentials
- **No enumerations found**: Check that domain models are loaded
- **TypeScript errors**: Ensure mendixplatformsdk and mendixmodelsdk are installed

### General

- **Module not found**: Run `npm install mendixplatformsdk mendixmodelsdk`
- **API key invalid**: Generate new API key from Mendix Developer Portal
- **Project ID**: Find in Mendix Portal under project settings
